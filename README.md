## frontend 

- peresentation layer

для запуска разработки и отладки перейти в каталог frontend и запустите vite
```
npm run dev
```
для сборки dist в проекте выполниете 
```
npm run build
```

для просмотра версии node
```
nvm list
```
для пересключения версии node
```
nvm use v16.20.2  
или
nvm use v20.10.0
```
